# Media Center

## Introduction

This repository helps you to setup a very good media
  center in an very easy way.

Services included are:

- Jackett
- Sonarr
- Radarr
- Transmission
- Plex

## How to run

Set the right path or `media/` will be used

```bash
./script/server
```

## Update

```bash
./script/update
```
